JRD Innovation Support Center Projects

We have completed several projects in various domains and right now working on several Projects

Articulated Robotic Arm

This project builds a life-sized wooden robotic arm with high load-bearing servos, combining mechanical design and advanced controls. It's perfect for learning and experimentation, suitable for students, hobbyists, and professionals exploring robotics and practical applications.

Workplace Management System

The project uses machine learning and Large Language Models (LLMs) to process brain signals, aiding neurodivergent individuals in speech and written communication. It integrates brain signal detection, ML processing, LLM synthesis, and IoT communication, enhancing tools for schools, employers, and associates.

Find out more →
Drone with Self-Made Flight Controller

This project develops affordable, scalable drones for large-scale surveillance at ports and airports. Using cost-effective hardware (coreless motors, ESCs) and software (NRF modules), it aims to enhance security measures, making drone technology accessible for Indian government agencies.

Find out more →
Advanced Accident Detection System (For 2-Wheeler Vehicle)

This project develops an automated distress signal system for two-wheeler accidents, using sensors and GPS to detect incidents and broadcast alerts to emergency services and nearby users. It aims to enhance road safety and promote community-driven emergency response.

Find out more →
Self Balancing Hexapod Robot

This project involves developing a six-legged walking robot with autonomous navigation and self-balancing capabilities. Integrating a camera system for object tracking will transform it from a pre-programmed machine into a versatile robot capable of navigating complex environments.

Find out more →
Novel Servo and Servo Motor

The NES project creates a custom servo library using machine learning to convert images into precise servo inputs, enhancing detection, tracking, and control. It focuses on improving latency and efficiency, with a second phase developing a seamless custom servo motor, advancing robotics and IoT image-based control systems.
